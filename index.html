// CARACTERÍSTICAS TÁCTICAS

01.

Arquitectura Asíncrona

Usa Goroutines en segundo plano. Lanza compiladores o comandos de Git sin pausar ni congelar tu buffer de escritura principal.

02.

Viewport Inteligente

Motor matemático de scroll vertical. Abre scripts ligeros o archivos kilométricos con un desplazamiento fluido integrado.

03.

Cero Distracciones

Controles estrictos por teclado y paneles limpios. Diseñado para optimizar tu enfoque y la memoria de tu máquina.

// STACK TECNOLÓGICO Y COMPATIBILIDAD

Lenguaje: Go (Golang) para garantizar binarios puros, velocidad nativa de ejecución y cero dependencias pesadas en el sistema.

TUI Engine: Termbox sustituido por tcell/v2 para un manejo de celdas UTF-8 robusto y control preciso de eventos.

Sistemas Operativos: Multiplataforma nativo. El núcleo compila limpiamente para Windows (CMD/PowerShell), Linux y macOS.

// Detalles de Compilación Cruzada para Windows

$env:GOOS="windows"

$env:GOARCH="amd64"

go build -ldflags="-s -w" -o johaedi.exe .

>>> Éxito: Binario estático compilado con stripping de símbolos.

[ DISTRIBUCIÓN DE CONSOLA ]

Instalación Directa

Sin ventanas pesadas de instalación gráfica. Copia y pega la tubería oficial correspondiente a tu sistema operativo directamente en tu terminal.

> Linux / macOS (Bash/Zsh):
curl -sS https://johaedi.dev/install.sh | sh
> Windows (PowerShell):
iwr -useb https://johaedi.dev/install.ps1 | iex
> Compilar desde código fuente (GitHub):

git clone https://github.com/johanmancebo/johaedi.git

cd johaedi

go build -ldflags="-s -w" -o johaedi
